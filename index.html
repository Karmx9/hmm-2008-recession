Hidden Markov Model: Real-Time Detection of the 2008 Financial Crisis

A 2-state HMM with manually specified parameters (A, B, π) infers the probability of an economic crisis state from two observable time series: WTI crude oil prices and residential mortgage delinquency rates. The Forward Algorithm updates the filtered posterior P(Crisis | O1:t) causally — using only observations available up to time t. Viterbi and Forward-Backward smoothing are computed over the full sequence and require future observations. Data: FRED MCOILWTICO and DRSFRMACBS, Jan 2004–Dec 2010.

Model note: HMM parameters (π, A, B) are manually specified with economic justification. This model was NOT estimated via Baum-Welch EM. The visualization demonstrates real-time detection (filtering) — not ex-ante prediction. Language "predicts" is intentionally avoided. Log-returns rt = ln(Pt/Pt-1) are used (not simple returns) because log-returns are time-additive and consistent with continuous compounding.
