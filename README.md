# HMM 2008 Recession Detection

**[Live Demo: hmm-2008-recession.vercel.app](https://hmm-2008-recession.vercel.app)**

This project is an animated visualization that uses a 2-state Hidden Markov Model to demonstrate how the 2008 Financial Crisis could have been detected in real time using only data available at that time. The model takes data from FRED which includes the monthly WTI crude oil prices along with quarterly residential mortgage delinquency rates tweaked to become monthly. This is then converted into discrete observation signals representing different market regimes. The visualization steps through all 83 months from February 2004 to December 2010 one step at a time, showing the exact mathematical recursion being computed at each step in a live panel. Three different algorithms were applied: the Forward filter, Forward-Backward smoothing, and the Viterbi algorithm. The HMM parameters were specified with macroeconomic theory and not just from data, which makes the modeling assumptions transparent. This was showcased as mainly a math project with a heavy emphasis on applied probability in real time markets.
