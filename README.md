# HMM 2008 Recession Detection

**[Live Demo: hmm-2008-recession.vercel.app](https://hmm-2008-recession.vercel.app)**

This project is an interactive, animated visualization that uses a 2-state Hidden Markov Model (HMM) to demonstrate how the 2008 financial crisis could have been detected in near real-time using only data available at the time. The model takes two observable economic time series from FRED — monthly WTI crude oil prices (MCOILWTICO) and quarterly residential mortgage delinquency rates (DRSFRMACBS), interpolated to monthly — and converts them into discrete observation signals representing regimes like "Financial Stress" or "Dual Stress." At each month, the Forward Algorithm updates a filtered posterior probability P(Crisis | observations so far), answering the question: given everything we have seen up to now, how likely is it that the economy is in a crisis state? The visualization steps through all 83 months from February 2004 to December 2010 one at a time, showing the exact mathematical recursion being computed at each step in a live trace panel. Three algorithms are implemented — the causal Forward filter, Forward-Backward smoothing (which uses the full sequence for hindsight comparison), and the Viterbi algorithm (which finds the single most probable hidden state path using log-domain arithmetic to prevent underflow). The HMM parameters (initial distribution π, transition matrix A, and emission matrix B) are manually specified with economic justification rather than estimated from data, making the modeling assumptions fully transparent. All mathematics is rigorous and correct, built as a teaching tool for a university probability course reviewed at the PhD level. The entire visualization runs as a single self-contained HTML file with no build step, using Chart.js for synchronized charts and inline SVG for the animated state diagram.
